## Installation

::: info
Require: PHP >= 7.1
:::

Installation is very simple as it is a standard [Composer](https://getcomposer.org/) package, which can be installed directly using the following command.

```bash
composer require zjkal/time-helper
```

## Update
```bash
composer update
```
